Interactive Image Window

Selection

LMB click - select just one object

CTRL + LMB - add current selected object to previous selection

selection by ImageJ - objects can be selected by polygon, circle and other rois in ImageJ

Configuration

Drawing configuration for image window can be changed in Interactive Analyzer Result > Edit > Options menu

Interactive Result Table

Selection

LMB CLICK - select one row

CTRL + LMB - add current selected row to previous selection

SHIFT + LMB - rows in clicked range are selected

Configuration

Configuration be changed in Interactive Analyzer Result > Edit > Options menu

Export

Table can be exported by Interactive Analyzer Result > Export menu

Object Filtering Frame

Selection

LMB click - select one shape

CTRL + LMB - add current selected shape to previous selection

DRAGGED CLICK - select all shapes which were crossed by mouse dragged

Configuration

Configuration be changed in Interactive Object Analyzer > Edit > Options

Export
